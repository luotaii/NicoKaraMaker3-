# NicoKaraMaker3 简体中文临时资源包

这是 NicoKaraMaker3 的非官方简体中文临时资源包。

本项目只用于在原作者正式加入简体中文翻译前，临时提供中文界面资源。等原作正式支持简体中文后，本项目将会归档，不再继续维护。

## 使用前准备

请先从 Microsoft Store 安装原版 NicoKaraMaker3。

本资源包不包含原软件本体，也不能单独运行。它只是为已经安装好的 NicoKaraMaker3 添加简体中文界面资源。

## 适用版本

当前资源包适用于：

`NicoKaraMaker3 13.43.158.0`

如果你的 NicoKaraMaker3 版本不同，可能无法安装，或者安装后无法正常显示中文。

## 安装方法

1. 下载 Release 中的压缩包。
2. 解压压缩包。
3. 双击运行：

   `双击安装_NicoKaraMaker3_中文精翻可用版.cmd`

4. 如果 Windows 弹出安全提示，请选择“仍要运行”。
5. 如果弹出管理员权限确认，请选择“是”。
6. 等安装窗口提示完成后，按回车关闭窗口。
7. 打开 NicoKaraMaker3。

安装成功后，界面会显示为简体中文。

## 恢复原版

如果想恢复原版界面，请双击运行：

`双击恢复_NicoKaraMaker3_原版.cmd`

恢复完成后，重新打开 NicoKaraMaker3 即可。

## 压缩包内容

- `NicoKaraMaker3_13.43.158.0_zh-CN_refined_stable.msix`
  - 简体中文资源包。
- `NicoKaraMaker3_zh_test.cer`
  - 安装测试资源包所需的证书公钥。
- `install_nicokara_zh_refined_stable.ps1`
  - 安装脚本。
- `双击安装_NicoKaraMaker3_中文精翻可用版.cmd`
  - 给普通用户使用的双击安装入口。
- `restore_nicokara_original.ps1`
  - 恢复原版脚本。
- `双击恢复_NicoKaraMaker3_原版.cmd`
  - 给普通用户使用的双击恢复入口。
